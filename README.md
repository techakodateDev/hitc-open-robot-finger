# hitc-open-robot-finger
<概要>
本リポジトリは、3Dプリントで造形可能な柔軟性を有するロボット用フィンガーを公開するものです。
個人利用・商用利用を問わず、自由に使用・改変・再配布が可能です。


<特徴>
・PLAなどの一般的な3Dプリント用フィラメントで造形可能な柔軟指
・一般的なFFF方式の3Dプリンタで出力可能


<ファイル構成>
/models
　/[モデル名]：3Dモデル（STL, OBJ, 3MF, STEP）,イメージファイル
/docs：補足資料


<使用方法>
/models - /[モデル名]内のファイルをダウンロードし、スライサーソフトに読み込んだ後、適宜条件を設定してプリントしてご利用ください。
造形方向はフィンガーモデルの側面を下方に向け、サポート無しでの造形を推奨します。


[推奨プリント条件]
・推奨材料	：PLA
・積層ピッチ	：0.2 mm
・サポート	：不要
・ブリム　	：必要に応じて
※0.5mmの薄板造形が含まれますので、外壁の造形を薄板対応の設定にするか、0.2mmのノズルを利用するなど、用途に応じて調整してください
